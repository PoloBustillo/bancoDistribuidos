# Next.js Frontend for Distributed Banking System

## Project Requirements
- **Type**: Next.js Application
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Purpose**: Educational UI to demonstrate distributed systems concepts

## Features
1. Multi-worker endpoint selection
2. User authentication (register/login)
3. Multiple account management
4. Account sharing with role-based access
5. Banking operations with distributed locks visualization
6. Individual card management
7. Real-time worker status display
